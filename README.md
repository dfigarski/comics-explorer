# Marvel Comics Explorer

## Opis projektu
Marvel Comics Explorer to prosta aplikacja internetowa stworzona przy użyciu React i Vite.
## Temat projektu
Biblioteka komiksów Marvel – eksplorator danych z API Marvel.

## Opis domeny aplikacji
Aplikacja służy do wyszukiwania i przeglądania komiksów Marvel.

## Użytkownik
Użytkownikiem aplikacji jest osoba zainteresowana komiksami Marvel.

## Problem
brak prostego i czytelnego panelu do przeglądania komiksów Marvel w jednym miejscu.

## Główna encja
Główną encją projektu jest komiks/film.

## Uruchomienie projektu
npm install
npm run dev

### Przykładowe pola encji:
- id
- title
- description
- thumbnail
- year

## Zrealizowane elementy

## Struktura projektu
- `src/components` – komponenty interfejsu
- `src/data` – przykładowe dane lokalne
- `src/models` – model danych
- `src/styles` – style CSS
- `src/App.jsx` – główny komponent aplikacji
- `src/main.jsx` – punkt startowy aplikacji
